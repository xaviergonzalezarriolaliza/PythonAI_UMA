## How to Use This Repository

1. Each Jupyter notebook (.ipynb) contains explanations, code examples, and exercises
2. Associated .zip files contain additional materials for that unit
3. Python scripts (.py) are standalone programs demonstrating specific concepts
4. Work through materials sequentially by unit number

## Notes

- Protected attributes are indicated with underscore prefix (e.g., _marca)
- Notebooks support interactive execution and experimentation
- All materials follow Spanish naming conventions (as per curriculum)
